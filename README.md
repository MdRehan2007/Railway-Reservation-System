
# 🚆 Railway Reservation System (C++)

## 📌 Description
This project is a console-based Railway Reservation System developed using C++.  
It allows users to book and cancel train tickets with automatic seat allocation.

## 🎯 Features
- View available trains
- Book tickets
- Cancel tickets
- Automatic seat/berth allocation based on age
- Different coach types (Sleeper, AC, General)
- Ticket generation with details
- Cancellation with refund calculation

## 🛠️ Technologies Used
- C++
- Object-Oriented Programming (OOP)
- STL (Vectors)
